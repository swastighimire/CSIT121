Understanding XML and JavaScript

What is XML?

XML (Extensible Markup Language) is a markup language that defines a set of rules for encoding documents in a format that is both human-readable and machine-readable.

How is XML used?

XML is commonly used for storing and transporting data across different systems and platforms. It is widely used in web services, configuration files, data interchange, and more.

What does XML look like? (Example)

        <bookstore>
          <book category="fiction">
            <title>Harry Potter</title>
            <author>J.K. Rowling</author>
            <year>2005</year>
            <price>29.99</price>
          </book>
        </bookstore>

What is JavaScript?

JavaScript is a high-level, interpreted programming language that is widely used for creating interactive and dynamic content on web pages.

How is JavaScript related to HTML and CSS?

JavaScript works alongside HTML and CSS to add functionality and interactivity to web pages. While HTML provides the structure, and CSS handles the presentation, JavaScript adds behavior to the elements on the page.

Common JavaScript Elements, Properties, Methods, and Functions

  • Element: document.getElementById()
  • Property: innerHTML
  • Method: getElementById().addEventListener()
  • Function: alert()
